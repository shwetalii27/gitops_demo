# GitOps Demo Project

## 📌 Project Title
GitOps-Based Continuous Delivery System

## 💡 Description
This project demonstrates GitOps by automatically deploying a website whenever changes are pushed to GitHub.

## ⚙️ Tools Used
- GitHub
- GitHub Actions
- GitHub Pages

## 🚀 How It Works
1. Developer pushes code to GitHub
2. GitHub Actions triggers deployment
3. Website updates automatically

## 👥 Team Roles
- Frontend Developer
- Git Manager
- CI/CD Engineer
- Deployment Manager
